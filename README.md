# Credit Risk Analysis

This project focuses on predicting credit risk using Machine Learning techniques in Python. The model is trained using customer banking data to classify whether a customer is likely to be a credit risk or not.

## Project Overview

The main objective of this project is to analyze loan-related data and build classification models for credit risk prediction. Different machine learning algorithms were used and evaluated based on their performance.

## Features

- Data preprocessing and cleaning
- Train-test data splitting
- Credit risk prediction
- Confusion matrix visualization
- Accuracy evaluation of models

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Models Used

- Logistic Regression
- Random Forest Classifier
- Support Vector Machine (SVM)

## Output

The project generates prediction results along with confusion matrix visualization for model evaluation.

## Dataset

The dataset contains banking and loan-related information used for training and testing the machine learning models.
